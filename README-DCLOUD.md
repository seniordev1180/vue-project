## 安装

```bash
# 克隆项目
git clone https://github.com/chuzhixin/vue-admin-beautiful.git
# 进入项目目录
npm vue-admin-beautiful
# 安装依赖
npm i
# 本地开发 启动项目
cnpm run serve
```
